### *今日重點*：
- 內建的function
- 內建的module
- 自訂的function
- *function分為會傳出值及不傳出值*
- **function的功能：**   
  1. 結構化程式碼
  2. 簡化邏輯思考的複雜度
  3. 可以重複呼叫
  4. 簡化主程式的程式碼
```
def fuction name (參數, 參數):
  變數
return 結果
```
呼叫funcion並**傳出值**：   
變數 = funcion(引數值, 引數值)

typehint 型別提示->None->不傳出值   
```
def funcion()->None:
```

讓程式碼執行主程式：
```
if __name__ == '__main__':
    main()
```
